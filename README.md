## How to Use

1. **Adding Tasks**: Type your task in the input field, select priority, and click "Add Task" or press Enter
2. **Completing Tasks**: Click the circular checkbox next to any task
3. **Editing Tasks**: Click the edit (pencil) icon to modify a task
4. **Deleting Tasks**: Click the delete (trash) icon to remove a task
5. **Filtering**: Use the filter buttons to view All, Pending, or Completed tasks
6. **Bulk Actions**: Use "Clear Completed" to remove all finished tasks
